# Code of Conduct / 行動規範

## 私たちの約束

このプロジェクトに参加するすべての人にとって、ハラスメントのない体験を提供することを約束します。年齢・体格・障害・民族・性自認や性表現・経験の程度・国籍・容姿・人種・宗教・性的指向にかかわらず、誰もが歓迎されます。

## 期待される行動

- 相手を尊重し、思いやりのある言葉を使う
- 異なる意見・視点・経験を尊重する
- 建設的なフィードバックを穏やかに受け入れる
- コミュニティにとって最善なことを優先する

## 許容されない行動

- 性的な言葉・イメージの使用、望まれない性的な言動
- 侮辱的・軽蔑的なコメント、個人攻撃や政治的攻撃
- 公私を問わないハラスメント
- 許可なく他者の私的情報（住所・メールアドレス等）を公開すること

## 適用範囲

この行動規範は、プロジェクトの空間（Issue、Pull Request、その他のやり取り）すべてに適用されます。

## 違反の報告

問題のある行動を見かけた場合は、リポジトリオーナー（[@tkm112345](https://github.com/tkm112345)）まで報告してください。報告はすべて速やかに、公正に対応されます。
